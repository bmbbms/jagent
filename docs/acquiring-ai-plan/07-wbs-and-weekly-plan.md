# 一期详细 WBS 与按周推进计划

## 一期详细 WBS

### WP1 项目启动与范围冻结

目标：先把边界定住，防止后面失控。

任务：

- 明确一期业务范围
- 确认试点部门、试点角色、试点用户
- 明确不纳入范围的场景
- 制定项目节奏和评审机制

交付物：

- 一期范围说明
- 试点名单
- 里程碑计划

### WP2 业务场景梳理

目标：把一期场景拆清楚。

任务：

- 梳理商户AI场景
- 梳理运营支持场景
- 梳理数据支持场景
- 定义每个场景的输入、输出、处理路径、升级条件

交付物：

- 场景清单
- 场景说明卡
- 优先级列表

### WP3 规则与流程沉淀

目标：把经验变成可执行规则。

任务：

- 梳理审核规则
- 梳理处理 SOP
- 梳理人工审批边界
- 梳理例外情况和回退逻辑

交付物：

- Skill 初稿
- Workflow 初稿
- 规则说明文档

### WP4 技术架构与工程初始化

目标：建立统一工程骨架。

任务：

- 初始化代码仓
- 建立目录结构和环境规范
- 搭建 Agno + FastAPI 基础框架
- 建立配置、日志、异常处理框架

交付物：

- 主工程骨架
- 环境部署说明
- 基础服务可运行

### WP5 权限、审批、审计底座

目标：先把治理能力打牢。

任务：

- 接入统一认证
- 设计 RBAC
- 建审批任务模型
- 建审计日志模型
- 定义高风险动作管控规范

交付物：

- 权限模型
- 审批模型
- 审计能力基础版

### WP6 Tool / MCP 接入

目标：把系统能力标准化暴露出来。

任务：

- 确定一期对接系统
- 编写 Tool 接口规范
- 接入只读查询 Tool
- 接入审批和写操作 Tool
- 增加异常、限流、超时处理

交付物：

- Tool Registry
- 一期核心 Tool
- 系统接入清单

### WP7 Knowledge / RAG 建设

目标：让问答和规则解释有依据。

任务：

- 梳理知识源
- 建立接入、清洗、切片、索引流程
- 建立权限过滤机制
- 建立答案引用和出处返回机制

交付物：

- 一期知识库
- 检索流程
- 引用规范

### WP8 Skill 体系建设

目标：形成可复用的业务能力单元。

任务：

- 统一 SKILL 模板
- 编写首批 Skills
- 评审和修订 Skills
- 建立版本和启停机制

交付物：

- Skills 仓库
- 首批可用 Skills
- Skill 管理规范

### WP9 Workflow 场景实现

目标：把高风险场景做成 AI 增强工作流。

任务：

- 实现调额审核流程
- 实现入网审核流程
- 实现变更审核流程
- 接入审批流和回退流

交付物：

- 一期 Workflows
- 审批流打通
- 回退机制

### WP10 前端工作台与试点验证

目标：确保业务真正能用。

任务：

- 搭建工作台
- 展示问答、引用、工具结果、审批状态
- 组织试点验证
- 收集反馈并优化

交付物：

- Web 工作台
- 试点反馈报告
- 优化清单

## 按周推进计划

### 第 1 周：项目启动

- 冻结一期范围
- 确定试点部门和角色
- 拉齐业务目标和边界
- 建项目例会和评审机制

### 第 2 周：场景与规则梳理

- 输出首批场景清单
- 梳理首批 Skills
- 梳理首批 Workflows
- 输出首批 Tool 清单和知识源清单

### 第 3 周：工程骨架和底座设计

- 初始化主工程
- 完成架构设计
- 建立权限、审批、审计模型
- 定义 Tool / Skill / Workflow 接入规范

### 第 4 周：底座开发

- 完成认证与 RBAC 基础版
- 完成审计日志基础版
- 完成 Tool Registry 基础版
- 完成知识库接入框架基础版

### 第 5 周：业务能力第一批接入

- 上线商户AI问答能力
- 接入商户资料、交易、风险查询 Tool
- 接入商户 FAQ 和制度知识库
- 完成首批 Merchant Skills

### 第 6 周：运营支持第一批接入

- 完成调额审核辅助
- 完成调额审核 Workflow
- 打通审批提交和审批回执
- 增加审核意见结构化输出

### 第 7 周：更多运营与数据能力

- 完成入网审核辅助
- 完成变更审核辅助
- 完成直营销售数据助手
- 打通固定报表查询

### 第 8 周：前端工作台联调

- 打通统一对话入口
- 展示知识引用和工具结果
- 展示审批状态和处理记录
- 完成试点可用版

### 第 9 周：试点运行

- 选定试点用户试用
- 收集问答准确率、审核命中率、转人工率
- 发现问题并修复
- 调整规则和技能内容

### 第 10 周：总结与二期规划

- 输出一期效果复盘
- 形成问题清单和优化计划
- 确定二期扩展范围
- 准备正式汇报材料
